# Missions and Objectives

## Scope Note
- This doc describes target mission structure.
- It is less important than movement, combat, and enemy-navigation docs for the current milestone.

## Mission Shape
- Short-to-medium runs with clear primary objective, optional side goals, and escalating risk regions.

## Objective Types
- Retrieve or secure target.
- Activate or defend sequence.
- Eliminate priority threats.
- Escort or timed extraction setup.

## Progression Interaction
- Optional objectives increase reward multipliers.
- Deeper mission stages increase loot quality and failure risk.
- Objective completion influences post-run progression payout.

## Co-op Requirements
- Objectives should permit role contribution from all classes.
- Mission state is host-driven and replicated via explicit objective events.
