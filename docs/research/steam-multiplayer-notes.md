# Steam Multiplayer Notes

## Purpose
- Capture the currently chosen external multiplayer direction.
- Keep this separate from the active singleplayer-first implementation milestone.

## Current Direction
- Use Steam Lobby for discovery and invites.
- Use Steam P2P with relay fallback for transport.
- Use a host-authoritative session model when multiplayer implementation starts.

## Status
- Research note only.
- Useful as a direction marker, not as a normative architecture document.
- Normative multiplayer decisions live in the ADRs and in [networking.md](d:/Game/DEV/iiWii/iiwii/docs/architecture/networking.md).
